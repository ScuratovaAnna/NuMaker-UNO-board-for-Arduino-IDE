enum  	Direction { DIRECTION_CCW = 0, DIRECTION_CW = 1 }
Индикатор направления Символические названия направления, в котором вращается двигатель.
DIRECTION_CCW Counter-Clockwise. --> Против часовой стрелки.
DIRECTION_CW Clockwise.          --> По часовой стрелке.

enum  	MotorInterfaceType {
  FUNCTION = 0, DRIVER = 1, FULL2WIRE = 2, FULL3WIRE = 3,
  FULL4WIRE = 4, HALF3WIRE = 6, HALF4WIRE = 8
}
Символические имена для количества выводов. 
Используйте это в аргументе контактов конструктора AccelStepper, чтобы предоставить символическое имя для количества используемых контактов.
FULL4WIRE 4 wire full stepper, 4 motor pins required  --> ПОЛНЫЙ 4ПРОВОД 4-проводной полный шаговый двигатель, требуется 4 контакта двигателя

setMaxSpeed (float speed) --> Устанавливает максимально допустимую скорость. Функция run() будет ускоряться до скорости,
 установленной этой функцией. Внимание: максимально достижимая скорость зависит от вашего процессора и тактовой частоты. 
По умолчанию maxSpeed составляет 1,0 шага в секунду.	
Параметры
(float speed) Желаемая максимальная скорость в шагах в секунду. Должно быть > 0. Внимание: Скорости, 
превышающие максимальную скорость, поддерживаемую процессором, могут Приводить к нелинейным ускорениям и замедлениям.

distanceToGo () --> Расстояние от текущей позиции до целевой позиции.
Возвращает
расстояние от текущего положения до целевого положения в шагах. Положительное значение по часовой стрелке от текущего положения.

setSpeed (float speed) --> Устанавливает желаемую постоянную скорость для использования с runSpeed().

runSpeed() --> Опросите двигатель и сделайте шаг, если требуется шаг, реализуя постоянную скорость, установленную последним вызовом setSpeed(). 
Вы должны вызывать это как можно чаще, но не реже одного раза за интервал шага
